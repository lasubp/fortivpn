# Shosrtcut scripts to connect to office fortivpn

Connect to Fortinet VPN using openfortivpn on background

## Requirements:
- ``` openfortivpn ``` installed

## Usage

Setup host, port and credentilas in vpn-up.sh

Run **`vpn-up.sh`** to connet to vpn
To kill VPN use **`vpn-kill.sh`** script
If logs needed they can be found in **/tmp/vpn-output.log** or by running script **`vpn-log.sh`**
